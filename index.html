Dataset
Primary Dataset
Titanic Dataset

The Titanic dataset contains passenger information from the RMS Titanic's maiden voyage. It includes features such as passenger class, age, fare, and survival status for 891 samples — a classic dataset for classification and predictive analysis tasks.

All five prompting techniques are applied to this same dataset, allowing direct comparison of how each strategy shapes the LLM's analytical output.
